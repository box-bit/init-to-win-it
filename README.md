# 🌿 MicroAdventure: Init-to-Win-It

Welcome to the **MicroAdventure** app, built for the Init-to-Win-It Hackathon. This mobile application is designed to encourage spontaneous exploration and nature connection through "microadventures"—short, accessible, and exciting local challenges.

## Wanna try it on your phone?
Download `5to9.apk` to your android phone <br>
Click on it and select install

## 🚀 The Concept

Microadventures are about finding adventure in your local area. Whether it's leaving your route to chance with a coin flip or identifying local flora, this app gamifies the world around you.

---

## ✨ Features

- **Mode Selection**: Choose your vibe—from "urban legend" "Chaos" "Survivalist".
- **Real-time Tracking**: Interactive coin flipping and progress tracking for active adventures.
- **Local Persistence**: Powered by **Expo SQLite** to save your adventure history, scores, and photo logs locally.
- **Leaderboard & Profile**: Track your stats and see how you rank against the "Exploration Level" of the community.
- **Custom UI**: A polished, immersive experience with linear gradients and custom tab navigation.

---

## 🛠 Tech Stack

- **Framework**: [Expo](https://expo.dev/) / [React Native](https://reactnative.dev/)
- **Navigation**: [React Navigation 7](https://reactnavigation.org/) (Tabs + Native Stack)
- **Database**: [Expo SQLite](https://docs.expo.dev/versions/latest/sdk/sqlite/) for local state persistence.
- **Icons/Assets**: Custom styled icons and theme-consistent imagery.
- **Storage**: [AsyncStorage](https://react-native-async-storage.github.io/async-storage/) for lightweight settings.

## 🏃‍♂️ Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (LTS)
- [Expo Go](https://expo.dev/client) app on your iOS or Android device.

### Installation

1. **Clone the repository**:
   ```bash
   git clone <repo-url>
   cd mobile
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npx expo start
   ```

4. **Scan the QR code**:
   Open the Expo Go app on your phone and scan the code shown in your terminal.
